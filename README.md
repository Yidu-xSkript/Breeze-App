# Breeze-App

Breeze App is a weather desktop app designed and developed using VueJs, VueX, Tailwind and ElectronJs. 
> Created By xSkript

#### Build Setup

```bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


```
